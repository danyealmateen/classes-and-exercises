JSON = JavaScript Object Notation

JSON tillåter följande JS datatyper:
*String
*Nummer
*Objekt
*Array
*Boolean
*Null

Tillåter inte: 
*Funktion
*Datum
*Undefined

*Måste skrivas med dubbla-citat tecken

API (application programming interface) = Kommunikation mellan system
*API dokumentation (kan ofta inkludera demonstrationer men inte alltid)
*Kan användas för att kontrollera externa system
*Kan skicka och returnera data
